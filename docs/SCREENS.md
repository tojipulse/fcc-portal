# SCREENS.md

> FCC PORTAL Screen Specification

---

# 1. 目的

このドキュメントでは FCC PORTAL の全画面を一覧化し、

- URL
- 利用権限
- 主な機能
- 遷移先

を管理します。

---

# 2. 権限一覧

|権限|説明|
|------|----------------|
|保護者|子どもの情報のみ利用可能|
|コーチ|全予定のコーチ出席登録可能|
|管理者|全機能利用可能|

---

# 3. ホーム

URL

```
/
```

権限

```
全ユーザー
```

機能

- 月間カレンダー
- 今日の予定
- お知らせ
- フッターメニュー

遷移

```
予定詳細
```

---

# 4. 予定詳細

URL

```
/schedule/[id]
```

権限

```
全ユーザー
```

表示

- 日時
- 場所
- 対象学年
- 持ち物
- 添付ファイル
- 出欠状況

---

# 5. 保護者一括出欠

URL

```
/attendance
```

権限

```
保護者
```

表示

予定カード一覧

回答

- 出席
- 欠席
- 未定

未選択

↓

未回答

---

# 6. コーチ一括出席

URL

```
/coach-attendance
```

権限

```
コーチ以上
```

特徴

- 個別画面なし
- 予定詳細へ遷移しない
- 背景色変更
- 一括回答専用

回答

- 出席
- 欠席
- 未定
