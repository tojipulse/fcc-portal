# 1. 目的

TODO.md は FCC PORTAL の未実装機能・改善事項・保留事項・将来構想を管理するドキュメントです。

GitHub Issues の代わりとなるプロジェクト管理台帳として利用します。

# 2. ステータス

|状態|説明|
|------|----------------|
|TODO|未着手|
|DOING|作業中|
|REVIEW|レビュー待ち|
|DONE|完了|
|HOLD|保留|

# 14. 管理方針

- 完了した項目は削除しない
- DONEへ移動する
- 保留はHOLDへ変更する
- 優先順位は定期的に見直す
